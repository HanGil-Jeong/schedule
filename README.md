## 📌 주요 기능
- 일정 생성
  - 할 일, 작성자명, 비밀번호, 작성일/수정일 생성 
  - 데이터 베이스에 저장
- 일정 조회
  - 전체 일정 조회
  - 선택 일정 조회 : scheduleId를 통해 원하는 일정 조회
- 일정 수정
  - 선택한 일정 수정 : 조회된 일정 정보를 비밀번호 입력 후 일치하면 수정 가능
- 일정 삭제
  - 선택한 일정 삭제 : 조회된 일정 정보를 비밀번호 입력 후 일치하면 삭제

 | 기능       | method   | URL | requset                                                                                                                        | response | 상태 코드      |
